# 🧪 QA Academy — Free QA Engineer Learning Platform

A complete, free, deployable learning platform for QA Engineers. 18 modules, 70+ lessons, 180 quiz questions.

## Tech Stack
- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **localStorage** for progress (no backend)

## Features
- 18 comprehensive modules (manual testing → full automation)
- Interactive quizzes with explanations
- Progress tracking with XP and streaks
- Dark/light mode
- Mobile responsive
- SEO optimized
- AdSense-ready placeholders

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

```bash
npm i -g vercel
vercel
```
